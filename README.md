# Sistema de Gestión Bancaria Familiar
Este proyecto es un simulador de dinero para el hogar hecho en Python. Sirve para crear perfiles de diferentes personas (como padres, hijos o asesores) y ver cómo manejan sus cuentas del banco, sus ahorros y sus deudas.

# Descripción
El sistema funciona como una maqueta digital que imita cómo se organiza una familia real y su economía. Se ha diseñado para que sea muy fácil de usar en cualquier computadora, ya que funciona con lo que ya trae Python de serie. Esto significa que no necesitas instalar programas extraños ni componentes adicionales para que todo funcione.

# Estructura del Proyecto
El programa se organiza en las siguientes partes:

## Cuenta Bancaria: Donde se controla cuánto dinero hay y se hacen los movimientos básicos.

## Perfiles de Usuario: Los distintos roles de la casa, como Adulto, Hijo, Padre o Madre Asesora.

## Funciones Especiales: La figura del Agente Financiero para ayudar con inversiones y trámites.

## Economía: Herramientas para llevar la cuenta de los ahorros y lo que se debe.

# Requisitos
Tener instalado Python 3.
No hace falta descargar nada extra (sin usar comandos de instalación).

# Instalación y Uso
Descarga todos los archivos del proyecto en una misma carpeta.
Abre y pon en marcha el archivo principal (main.py).
